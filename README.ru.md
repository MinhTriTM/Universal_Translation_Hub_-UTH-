## Обзор

**Universal Translation Hub (UTH)** — это мультиагентная AI-система для автоматического перевода **3 типов мультимедийного контента**:

| Конвейер | Вход | Выход |
|----------|------|-------|
| **Перевод игр** | Файлы игр (12 движков: RPG Maker, Unity, Ren'Py, Kirikiri...) | Полностью локализованная игра на вьетнамский |
| **Перевод манги** | Изображения манги/комиксов (JPG, PNG, PDF) | Переведённые изображения с рендерингом текста |
| **Озвучка фильмов** | Видео + субтитры (MKV, MP4, ASS, SRT) | Видео с AI-озвучкой на вьетнамский |

### Зачем этот проект?

> **Проблема:** Миллионы игр, манги и фильмов из Японии, Китая и Кореи не имеют вьетнамского перевода. Ручной перевод занимает месяцы, дорого и не масштабируется.

> **Решение:** UTH использует **10 AI-субагентов**, работающих автоматически на базе **Xiaomi MiMo V2.5**, для перевода любого мультимедийного контента на вьетнамский за минуты.

## Архитектура системы

```
┌─────────────────────────────────────────────────────────────────┐
│                    ДИРЕКТОР-АГЕНТ                                │
│              (MiMo-V2.5-Pro — Флагманский reasoning)            │
│   Анализ запроса → Выбор конвейера → Координация 9 субагентов  │
└──────────┬──────────────┬──────────────┬────────────────────────┘
           │              │              │
  ┌────────▼────────┐ ┌──▼──────────┐ ┌▼──────────────┐
  │  КОНВЕЙЕР ИГР  │ │ КОНВЕЙЕР    │ │  КОНВЕЙЕР     │
  │  (12 движков)   │ │ МАНГИ      │ │  ФИЛЬМОВ      │
  │                 │ │(OCR+Рендер) │ │  (STT+TTS)    │
  └─────────────────┘ └─────────────┘ └───────────────┘
```

### 10 AI-субагентов

| # | Агент | Модель MiMo | Роль |
|---|-------|-----------|------|
| 1 | **Директор** | MiMo-V2.5-Pro | Оркестрация, рассуждения, выбор конвейера |
| 2 | **Маршрутизатор** | MiMo-V2.5 | Классификация ввода → игра/манга/фильм |
| 3 | **Переводчик** | MiMo-V2.5 | Многоязычный перевод (ZH/JA/KO/EN → VI) |
| 4 | **OCR-агент** | MiMo-V2.5 | Распознавание текста на изображениях |
| 5 | **Агент инпейнтинга** | MiMo-V2.5 | Удаление оригинального текста |
| 6 | **Агент рендеринга** | MiMo-V2.5 | Рендер переведённого текста |
| 7 | **STT-агент** | MiMo-V2.5 | Распознавание речи |
| 8 | **Голосовой агент** | MiMo-V2.5-TTS | Базовый TTS |
| 9 | **Клонирование голоса** | MiMo-TTS-VoiceClone | Клонирование голоса персонажей |
| 10 | **Агент QA** | MiMo-V2.5 | Оценка качества перевода |

