## 概述

**Universal Translation Hub (UTH)** 是一个多智能体AI系统，可自动翻译**3种多媒体内容**：

| 管道 | 输入 | 输出 |
|------|------|------|
| **游戏翻译** | 游戏文件（12种引擎：RPG Maker、Unity、Ren'Py、Kirikiri等） | 完全越南语化的游戏 |
| **漫画翻译** | 漫画图片（JPG、PNG、PDF） | 翻译后的图片（含渲染文字） |
| **影视配音** | 视频+字幕（MKV、MP4、ASS、SRT） | 带AI越南语配音的视频 |

### 为什么需要这个项目？

> **问题：** 来自日本、中国和韩国的数百万游戏、漫画和电影缺少越南语翻译。人工翻译耗时数月，成本高昂，且无法规模化。

> **解决方案：** UTH使用**10个AI子智能体**自动协作，由**小米MiMo V2.5**驱动，可在几分钟内将任何多媒体内容翻译成越南语。

## 系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                      总监智能体                                  │
│              (MiMo-V2.5-Pro — 旗舰推理模型)                      │
│   分析请求 → 选择管道 → 协调9个子智能体                          │
└──────────┬──────────────┬──────────────┬────────────────────────┘
           │              │              │
  ┌────────▼────────┐ ┌──▼──────────┐ ┌▼──────────────┐
  │   游戏管道      │ │  漫画管道   │ │   影视管道     │
  │  (12种引擎)     │ │(OCR+渲染)   │ │  (STT+TTS)    │
  └─────────────────┘ └─────────────┘ └───────────────┘
           │              │              │
  ┌────────▼──────────────▼──────────────▼────────────────────────┐
  │                    共享服务层                                   │
  │  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌─────────────────┐  │
  │  │统一翻译器│ │翻译记忆库│ │术语管理器│ │ MiMo语音引擎    │  │
  │  └──────────┘ └──────────┘ └──────────┘ └─────────────────┘  │
  └───────────────────────────────────────────────────────────────┘
```

### 10个AI子智能体

| # | 智能体 | MiMo模型 | 职责 |
|---|--------|---------|------|
| 1 | **总监** | MiMo-V2.5-Pro | 编排、推理、管道选择 |
| 2 | **路由器** | MiMo-V2.5 | 分类输入 → 游戏/漫画/影视 |
| 3 | **翻译器** | MiMo-V2.5 | 多语言翻译（中/日/韩/英 → 越南语） |
| 4 | **OCR智能体** | MiMo-V2.5 | 图片文字识别（漫画）和视频字幕识别 |
| 5 | **修复智能体** | MiMo-V2.5 | 删除原文、恢复背景 |
| 6 | **渲染智能体** | MiMo-V2.5 | 将翻译文字渲染到图片中 |
| 7 | **语音识别** | MiMo-V2.5 | 语音转文字 |
| 8 | **语音智能体** | MiMo-V2.5-TTS | 基础TTS — 字幕朗读、旁白 |
| 9 | **声音克隆** | MiMo-TTS-VoiceClone | 角色声音克隆 |
| 10 | **质量检查** | MiMo-V2.5 | 翻译质量评估 |

### 使用的模型（8/8）

| 模型 | 用途 | 预估月消耗Credits |
|------|------|-----------------|
| **MiMo-V2.5-Pro** | 总监智能体 — 复杂推理 | ~5000万 |
| **MiMo-V2.5** | 翻译、OCR、质量检查、路由、修复、渲染、语音识别 | ~2亿 |

## 当前状态

UTH基于**3个已有项目**构建：

| 来源项目 | 状态 | 对UTH的贡献 |
|---------|------|------------|

| **manga-image-translator** | 稳定版 | 漫画管道、OCR、修复、渲染 |

## 快速开始

```bash
git clone https://github.com/YOUR_USERNAME/universal-translation-hub.git
cd universal-translation-hub
pip install -r requirements.txt
cp .env.example .env
# 编辑 .env: MIMO_API_KEY=your_key_here
python main.py
```
