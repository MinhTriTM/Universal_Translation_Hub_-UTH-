## 5. Управление рисками

### 5.1 Реестр рисков

| ID | Риск | Вероятность | Влияние | Приоритет | Стратегия |
|----|------|-------------|---------|-----------|-----------|
| R-01 | Недостаточно VRAM для всех моделей | Высокое | Критическое | P0 | Квантизация, ленивая загрузка |
| R-02 | MiMo не поддерживает вьетнамский TTS | Среднее | Высокое | P1 | Fallback на другой TTS-движок |
| R-03 | Низкое качество OCR для манги | Среднее | Высокое | P1 | Fine-tuning, комбинация моделей |
| R-04 | Несовместимость игровых движков | Среднее | Среднее | P2 | Плагинная архитектура |
| R-05 | Перегрузка GPU при параллельных задачах | Высокое | Среднее | P1 | Очередь задач, ограничение параллелизма |
| R-06 | Сбои VoiceClone | Низкое | Среднее | P2 | Fallback на стандартный TTS |
| R-07 | Медленный инференс на слабом GPU | Высокое | Высокое | P1 | Квантизация INT4, батчинг |
| R-08 | Потеря данных при сбое | Низкое | Критическое | P0 | Транзакции, WAL-mode SQLite |
| R-09 | Несовместимость CUDA-версий | Среднее | Высокое | P1 | Документация требований |
| R-10 | Сроки разработки сорваны | Среднее | Среднее | P2 | MVP-подход, приоритизация |

### 5.2 План реагирования на риски

#### R-01: Недостаточно VRAM

```
УСЛОВИЕ: Использование VRAM > 90%

ДЕЙСТВИЕ:
  1. Активировать квантизацию INT8 (приоритет)
  2. Выгрузить неиспользуемые модели
  3. Последовательная обработка вместо параллельной
  4. Уведомить пользователя

ПОРОГИ:
  - Предупреждение: VRAM > 75%
  - Критический: VRAM > 90%
  - Аварийный: VRAM > 95% → принудительная выгрузка
```

#### R-05: Перегрузка GPU

```
УСЛОВИЕ: Очередь задач > max_concurrent_tasks

ДЕЙСТВИЕ:
  1. Поставить задачу в очередь
  2. Проверить приоритет задачи
  3. Выполнить при восстановлении ресурсов
  4. Уведомить пользователя о времени ожидания
```

### 6.2 Юнит-тесты

```python
# tests/unit/test_translator.py
import pytest
from agents.translator import TranslatorAgent
from agents.base import AgentMessage, AgentResponse

@pytest.fixture
async def translator():
    agent = TranslatorAgent()
    await agent.initialize()
    return agent

@pytest.mark.asyncio
async def test_translate_simple_text(translator):
    message = AgentMessage(
        sender="director",
        receiver="translator",
        action="translate",
        payload={
            "text": "Hello, world!",
            "source_lang": "en",
            "target_lang": "vi"
        }
    )
    response = await translator.process(message)
    assert response.success is True
    assert response.data["translated_text"] != ""
    assert response.metrics["latency_ms"] > 0

@pytest.mark.asyncio
async def test_translate_empty_text(translator):
    message = AgentMessage(
        sender="director",
        receiver="translator",
        action="translate",
        payload={"text": "", "source_lang": "en", "target_lang": "vi"}
    )
    response = await translator.process(message)
    assert response.success is False
    assert "пустой текст" in response.error.lower()

@pytest.mark.asyncio
async def test_translate_with_context(translator):
    message = AgentMessage(
        sender="director",
        receiver="translator",
        action="translate",
        payload={
            "text": "Use the Force, Luke!",
            "source_lang": "en",
            "target_lang": "vi",
            "context": "Star Wars, character: Luke Skywalker"
        }
    )
    response = await translator.process(message)
    assert response.success is True
    # Проверяем, что контекст учтён
    assert response.data["confidence"] >= 0.7
```
